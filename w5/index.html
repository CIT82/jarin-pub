HTML

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It structures the content on the web, providing a basic structure for web documents.

CSS

CSS (Cascading Style Sheets) is used for styling HTML documents. It allows you to control the layout and appearance of multiple web pages simultaneously.

Bootstrap

Bootstrap is a popular front-end framework that simplifies web development. It provides ready-to-use components and styles for creating responsive and visually appealing websites.

JavaScript

JavaScript is a versatile scripting language that adds interactivity to web pages. It enables dynamic content, form validation, and other client-side functionalities.
